# CODSOFT – Internship Projects

This repository contains three simple Python projects designed for learning and practice. Each project is independent and has its own folder with source code and README.md.

## Projects Included

### 1. [Task1 – To-Do List](Task1/README.md)
A command-line application to manage and organize daily tasks.  
**Features:** Add, view, complete, and delete tasks.

### 2. [Task2 – Calculator](Task2/README.md)
A simple calculator to perform basic arithmetic operations.  
**Features:** Addition, subtraction, multiplication, division, and error handling.

### 3. [Task3 – Password Generator](Task3/README.md)
Generates secure random passwords with letters, numbers, and special characters.  
**Features:** User-defined length, secure password generation, CLI interface.

## How to Run
1. Make sure Python 3.x is installed on your system.
2. Navigate to the folder of the project you want to run.
3. Run the corresponding Python file using:
```bash
python <filename>.py
